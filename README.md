## English Description

This project is a mobile and web-based medication tracking application developed using Flutter and Firebase Realtime Database.
Users can schedule medications, monitor intake times, and track medication status in real time.

The system supports:

* Real-time data synchronization

* Medication scheduling

* Intake status monitoring

* Cross-platform support (mobile & web)

##  Türkçe Açıklama

Bu proje, Flutter ve Firebase Realtime Database kullanılarak geliştirilmiş mobil ve web tabanlı bir ilaç takip uygulamasıdır.
Kullanıcılar ilaç saatlerini planlayabilir, ilaç alım durumlarını izleyebilir ve verileri gerçek zamanlı olarak takip edebilir.

Sistem aşağıdaki özellikleri destekler:

* Gerçek zamanlı veri senkronizasyonu

* İlaç zamanlama

* İlaç alım durumu takibi

* Mobil ve web platform desteği

##  Technologies / Teknolojiler

* Flutter

* Dart

* Firebase Realtime Database
